# Rock vs Mine Classifier

A Streamlit web app that classifies SONAR signals as Rock or Mine using an SVM model.

## Features
- Paste 60 SONAR values
- Predict Rock or Mine
- Model evaluation (Accuracy, Confusion Matrix, Classification Report)

## How to run
streamlit run sa.py
